Clean day. Sandbox dropped the session on the pre-made `claude/sharp-gauss-o6wdth`
work branch as expected; Step 0's `git checkout -B main HEAD` reattached to main
and plain `git push origin main` worked first try. Playwright ran on the
pre-staged Chromium. No TodoWrite tool in this harness — tracked runbook steps
manually; no impact.

One trap worth recording: locked theme.css gives every `<a>` a 44px
min-width/min-height (right for text links). Making the around-view door an
anchor swelled its drawn 24×40 (2x) panel to 44×44 — caught because the
interaction test asserted the bounding box. Fix: `min-width/min-height: 0` on
`.side-cabin__door`, with the 44px tap target carried by an invisible
`::after` pad (effective 48×56 at 2x, 60×76 at 3x). Also: the door anchor had
to move OUTSIDE the `role="img"` cabin container — children of `role="img"`
are presentational, so a link inside it would vanish from the accessibility
tree. It is now positioned in scene coordinates over the same pixels
(alignment asserted at both scales).
